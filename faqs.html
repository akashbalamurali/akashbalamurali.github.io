Troubleshooting

I see a blank next to Activity name when I try to add an event

This will happen if you don't have any activities defined in Prodler yet. You can add an activity using the + sign to the right of the Activity name entry or go to the Settings tab and add all the activities that you would like to track there. Once it is done, you should start seeing the activity names show up when you try to add an event.
